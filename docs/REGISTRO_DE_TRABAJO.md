# Registro de Trabajo del Equipo

**Propósito:** Usar este archivo para comunicar en qué está trabajando cada miembro del equipo. Antes de empezar una tarea importante, añade una entrada aquí para evitar conflictos y duplicación de trabajo.

---

## Formato de Entrada

- **Nombre:** [Tu Nombre]
- **Fecha:** [DD-MM-AAAA]
- **Tarea:** [Descripción breve de lo que vas a hacer. Ej: "Corregir el bug del login", "Implementar la subida de imágenes de perfil"]
- **Estado:** [Pendiente / En Progreso / Finalizado / Bloqueado]
- **Archivos Afectados (opcional):** [Lista de los principales archivos que planeas modificar]
